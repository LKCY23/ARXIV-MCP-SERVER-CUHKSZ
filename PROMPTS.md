# ArXiv MCP Server - Prompts 功能说明

本文档列出了所有可用的 MCP Prompts 及其功能。

## 📋 所有可用的 Prompts

### 1. `deep-paper-analysis` - 深度论文分析

**功能描述**：对单篇 arXiv 论文进行全面的深度分析

**参数**：
- `paper_id` (必填): arXiv 论文 ID，例如 "2401.12345"

**使用场景**：
- 需要深入理解某篇论文的贡献、方法、结果
- 准备论文综述或报告
- 评估论文的实际应用价值

**分析内容包括**：
- ✅ **执行摘要**：论文核心贡献的 2-3 句话总结
- ✅ **研究背景**：研究领域、问题定位、相关工作
- ✅ **方法论分析**：方法步骤、创新点、理论基础、实现细节、可复现性
- ✅ **结果分析**：实验设置、主要结果、统计有效性、与 SOTA 对比
- ✅ **实际应用**：如何实现和应用、所需资源、可用代码/数据集
- ✅ **理论意义**：对理论理解的推进、新概念、开放问题
- ✅ **未来方向**：局限性、后续研究方向、长期研究议程
- ✅ **更广泛影响**：社会、伦理、政策影响，实际应用前景

**示例使用**：
```json
{
  "name": "deep-paper-analysis",
  "arguments": {
    "paper_id": "2401.12345"
  }
}
```

### 2. `research-discovery` - 研究探索

**功能描述**：开始对特定研究主题的探索和发现

**参数**：
- `topic` (必填): 研究主题或问题，例如 "transformer architecture" 或 "multi-agent systems"
- `expertise_level` (可选): 用户熟悉程度，可选值：`beginner` / `intermediate` / `expert`
- `time_period` (可选): 搜索时间范围，例如 "2023-present" 或 "2020-2024"
- `domain` (可选): 学术领域，例如 `computer_science` / `physics` / `biology`

**使用场景**：
- 探索新的研究领域
- 了解某个主题的研究现状
- 找到入门和深入学习的方向
- 根据经验水平获得定制化的指导

**输出内容包括**：
- 📚 **主题概览**：研究领域简介、重要性、关键问题
- 📚 **基础工作**：建立该领域的早期关键论文、核心概念和理论
- 📚 **研究现状**：最新发展和趋势、主要研究方向、重要近期论文
- 📚 **关键主题和方法**：常见方法论和技术、不同学派或方法、方法比较
- 📚 **开放问题和挑战**：未解决的问题、当前方法的局限性、未来研究方向
- 📚 **推荐论文**：必读的基础论文、重要近期贡献、综述论文
- 📚 **入门指南**：建议阅读顺序、所需先修知识、深入探索的下一步

**示例使用**：
```json
{
  "name": "research-discovery",
  "arguments": {
    "topic": "graph neural networks",
    "expertise_level": "beginner",
    "time_period": "2022-present",
    "domain": "computer_science"
  }
}
```

### 3. `literature-synthesis` - 文献综述

**功能描述**：综合多篇论文的发现，生成结构化综述

**参数**：
- `paper_ids` (必填): arXiv 论文 ID 列表（逗号分隔），例如 "2401.12345,2402.23456,2403.34567"
- `synthesis_type` (可选): 综合类型，可选值：
  - `themes`: 按主题综合
  - `methods`: 按方法综合
  - `timeline`: 按时间线综合
  - `gaps`: 识别研究空白
  - `comprehensive`: 全面综合（默认）
- `domain` (可选): 学术领域，例如 `computer_science` / `physics` / `biology`

**使用场景**：
- 撰写文献综述
- 比较多篇论文的方法和结果
- 识别研究趋势和空白
- 整合相关研究领域的知识

**综合类型说明**：
- **themes（主题）**：按共同主题、话题或研究问题组织论文
- **methods（方法）**：按方法论、技术或方法分类论文
- **timeline（时间线）**：按时间顺序展示研究的演变
- **gaps（空白）**：识别研究空白和未探索的领域
- **comprehensive（全面）**：结合主题、方法和时间线的多维度综合

**输出内容包括**：
- 📊 **执行摘要**：论文概览、主要发现和结论
- 📊 **综合分析**：根据综合类型进行组织（主题/方法/时间线/空白/全面）
- 📊 **跨论文的主要发现**：共同发现和共识点、不同观点或矛盾、个别论文的突出贡献
- 📊 **比较分析**：不同方法的优缺点、论文之间的关系和依赖、论文如何相互建立或挑战
- 📊 **综合结论**：识别的整体模式和趋势、综合得出的关键见解、对研究领域的影响
- 📊 **研究空白和未来方向**：未解答的问题、需要更多研究的领域、有前景的未来方向
- 📊 **参考文献和论文详情**：每篇论文的关键贡献摘要、每篇论文如何融入更广阔的图景

**示例使用**：
```json
{
  "name": "literature-synthesis",
  "arguments": {
    "paper_ids": "2401.12345,2402.23456,2403.34567",
    "synthesis_type": "themes",
    "domain": "computer_science"
  }
}
```

### 4. `research-question` - 研究问题生成

**功能描述**：基于现有文献提出研究问题

**参数**：
- `paper_ids` (必填): arXiv 论文 ID 列表（逗号分隔），例如 "2401.12345,2402.23456"
- `topic` (必填): 研究主题或问题，例如 "reinforcement learning in multi-agent systems"
- `domain` (可选): 学术领域，例如 `computer_science` / `physics` / `biology`

**使用场景**：
- 基于已有文献寻找新的研究方向
- 识别研究空白并提出问题
- 规划研究项目
- 为论文或提案生成研究问题

**输出内容包括**：
- ❓ **研究背景**：研究主题摘要、现有文献概览、关键发现和当前知识状态
- ❓ **文献空白分析**：现有论文已建立的内容、仍未知或不确定的内容、当前研究的局限性和空白、未解决的矛盾或争论
- ❓ **制定的研究问题**：
  - 主要研究问题
  - 次要或支持性问题
  - 每个问题包括：问题本身（清晰陈述）、为何重要、如何解决已识别的空白、与现有文献的关系、解决该问题的潜在方法
- ❓ **问题优先级**：最需要首先解决的关键问题、会产生最大影响的问题、可行性考虑、问题之间的相互依赖
- ❓ **研究方向**：解决这些问题的潜在方法论、所需资源和考虑因素、预期结果和贡献、解决这些问题如何推进该领域
- ❓ **相关研究机会**：分析过程中出现的其他问题、未来调查的问题、与其他研究领域的联系

**示例使用**：
```json
{
  "name": "research-question",
  "arguments": {
    "paper_ids": "2401.12345,2402.23456",
    "topic": "graph neural networks for molecular property prediction",
    "domain": "computer_science"
  }
}
```

## 🔄 Prompt 使用流程

### 1. 列出可用 Prompts
客户端可以调用 `list_prompts()` 来查看所有可用的 prompts。

### 2. 获取 Prompt
客户端调用 `get_prompt(name, arguments)` 来获取具体 prompt 的提示词消息。

### 3. 使用 Prompt
客户端将获取到的提示词消息发送给 AI（如 Claude），AI 会根据提示词中的指令，调用相应的工具（如 `search_papers`、`read_paper` 等）来完成任务。

## 📝 注意事项

1. **参数验证**：所有必填参数都必须提供，否则会返回错误
2. **工具依赖**：Prompts 会指导 AI 使用相应的工具，确保工具可用
3. **上下文保持**：某些 prompts（如 `deep-paper-analysis`）会保持上下文，引用之前分析过的论文
4. **自定义调整**：根据 `expertise_level` 和 `domain` 等参数，prompts 会调整输出内容的深度和专业性

## 🔗 相关工具

所有 prompts 都会使用以下工具：
- `search_papers`: 搜索 arXiv 论文
- `download_paper`: 下载论文
- `read_paper`: 读取论文内容
- `list_papers`: 列出已下载的论文
